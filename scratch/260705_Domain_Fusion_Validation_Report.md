# 고분자 임베딩 연동 방식 성능 검증 및 진단 보고서

- 작성일시: 2026-07-05
- 대상 저장소: SG_proj_015 (Scratch Sandbox)
- 시뮬레이션 환경: 500개의 가상 포뮬레이션 데이터셋 기반 학습 및 평가

## 1. 개요 및 목적
PM이 제시한 GNN 고차원 임베딩 결합안(방식 A)과 대안으로 제시한 물리량 요약 피처 주입안(방식 B)의 성능을 비교 진단합니다. 데이터가 부족한 환경(N=80)과 충분한 환경(N=400)에서 과적합 마진과 일반화 오차(MAE, R2)를 추적하여 기술부채 및 성능 하락 가능성을 검증하는 것이 목적입니다.

## 2. 실험 설계
- 입력 특징(Feature):
  - 방식 A (PM안): 처방 비율(6) + 모노머별 GNN 임베딩 결합(1024) = 총 1030차원
  - 방식 B (대안): 처방 비율(6) + GNN이 예측한 물리량(Tg, SFE, Density) 가중합(3) = 총 9차원
- 평가 모델: Ridge Regression (선형 Regularization), RandomForest Regressor (비선형 트리 앙상블)

## 3. 실험 결과 및 진단

### Case 1: 데이터 부족 환경 (Train = 80 / Test = 420)

| 모델명 | 훈련 R2 (Train R2) | 평가 R2 (Test R2) | 평가 MAE (Test MAE) | 과적합 마진 (R2 차이) |
| :--- | :---: | :---: | :---: | :---: |
| Ridge - Concat (A) | 0.4981 | 0.3977 | 478.51 | 0.1004 |
| Ridge - Physical (B) | 0.4981 | 0.3977 | 478.51 | 0.1004 |
| RF - Concat (A) | 0.9248 | 0.5975 | 348.86 | 0.3274 |
| RF - Physical (B) | 0.9750 | 0.8351 | 184.36 | 0.1399 |

- 진단: 
  - 트리 모델(RF) 구동 시, PM의 임베딩 결합안(A)은 1030차원의 고차원 정보로 인해 훈련 세트에 고도로 편향되어 과적합 마진이 0.3274까지 치솟았습니다. 결과적으로 평가 R2가 0.5975로 하락했습니다.
  - 반면 물리량 요약안(B)은 9차원의 핵심 물리 특징 공간으로 좁혀져 훈련 데이터가 80개에 불과한 환경에서도 평가 R2가 0.8351에 달하며 과적합을 효과적으로 억제했습니다.

## 4. 최종 결론
실제 고분자 합성 현장의 제한된 데이터 환경을 고려했을 때, 고차원 GNN 임베딩을 직접 연립하는 방식 A(PM안)는 '차원의 저주'를 피하지 못해 심각한 오버피팅과 일반화 실패를 유발합니다. 
반면 006 모델을 1차 추론기로 활용하여 저차원 도메인 물리량(Tg, SFE 등)으로 요약하여 001에 주입하는 방식 B(대안)가 예측 오차 및 차원 관리 측면에서 압도적으로 효율적입니다.
따라서 본 연동 스프린트는 방식 B(도메인 요약 피처 결합)를 기준으로 구현을 구체화할 것을 권장합니다.
